Wireframes

Voordat we begonnen met de development hebben we Wireframes gemaakt, deze zijn gemaakt op basis van o.a. twee factoren. Namelijk de Meterstandpagina van Oasen, maar ook op basis van de Guidelines die we hieronder beschrijven. De Wireframes kunt u vinden op Figma. Een deel kunt u zien via de afbeelding hiernaast.

Research

Allereerst hebben we gekeken naar goede guidelines die we konden toepassen. Hier hebben we de A11Y guidelines toegepast. We hebben alle regels opgeschreven die voor ons van toepassing, en aan de hand daarvan zijn we gaan ontwerpen. We hebben deze wireframes gemaakt. Uiteindelijk hebben we deze ook omgezet in een prototype.


Programming & Code

Onze geprogrammeerde code is te vinden op Github. Deze zal voor iedereen beschikbaar zijn om te bekijken. Deze specifieke code is geprogrammeerd in HTML,Javascript en CSS. Veel succes met uw meterstand invullen.
